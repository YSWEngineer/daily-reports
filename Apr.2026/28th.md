## 模範解答
```ruby
#!/usr/bin/env ruby
# frozen_string_literal: true

require 'optparse'

def main
  option = parse_option
  contents_by_file_name = build_contents_by_file_name
  count_data_list = build_count_data_list(contents_by_file_name)
  count_data_list << build_total_numbers(count_data_list) if count_data_list.size > 1
  display_results(count_data_list, option)
end

def parse_option
  option = ARGV.getopts('l', 'w', 'c')
  no_option = { 'l' => true, 'w' => true, 'c' => true }
  option.value?(true) ? option : no_option
end

def build_contents_by_file_name
  if ARGV.size.zero?
    content = $stdin.read
    { '' => content }
  else
    contents = ARGV.map { |file_name| File.read(file_name) }
    ARGV.zip(contents).to_h
  end
end

def build_count_data_list(contents_by_file_name)
  contents_by_file_name.map do |file_name, file_content|
    {
      line: count_line(file_content),
      word: count_word(file_content),
      byte: count_byte(file_content),
      file: file_name
    }
  end
end

def count_line(file_content)
  file_content.split("\n").size
end

def count_word(file_content)
  file_content.split(' ').size
end

def count_byte(file_content)
  file_content.bytesize
end

def build_total_numbers(count_data_list)
  {
    line: count_data_list.sum { |hash| hash[:line] },
    word: count_data_list.sum { |hash| hash[:word] },
    byte: count_data_list.sum { |hash| hash[:byte] },
    file: 'total'
  }
end

def display_results(count_data_list, option)
  count_data_list.each do |result|
    print format_value(result[:line]) if option['l']
    print format_value(result[:word]) if option['w']
    print format_value(result[:byte]) if option['c']
    puts " #{result[:file]}"
  end
end

def format_value(value)
  value.to_s.rjust(8)
end

main
```

### 3. mainメソッド（プログラム全体の流れ）
```ruby
def main

  count_data_list << build_total_numbers(count_data_list) if count_data_list.size > 1
```

- `def main`：プログラム全体の処理の流れを担う、「司令塔」のようなメソッド
- `option = parse_option`：**オプションの解析**。`parse_option`メソッドを呼び出し、ユーザーが指定したオプション（`-l`, `-w`, `-c`）を取得し、変数`option`に代入する
- `contents_by_file_name = build_contents_by_file_name`：**ファイル内容の取得**。`build_contents_by_file_name`メソッドを呼び出し、ファイル名とその内容を紐付けたハッシュを取得する
- `count_data_list = build_count_data_list(countents_by_file_name)`：**カウントデータの作成**。各ファイルの行数・単語数・バイト数を計算したハッシュの配列を作成する
- `count_data_list << build_total_numbers(count_data_list) if count_data_list.size > 1`：**合計行の追加**。ファイルが2つ以上ある場合のみ、合計（total）を計算して`count_data_list`に追加する。`<<`は配列の末尾に要素を追加する演算子
- `display_results(count_data_list, option)`：**結果の表示**。カウントデータとオプション情報をもとに、結果を画面に表示する。

### 4. parse_optionメソッド（メソッド解析）
```ruby
def parse_option
  option = ARGV.getopts('l', 'w', 'c')
  no_option = { 'l' => true, 'w' => true, 'c' => true }
  option.value?(true) ? option : no_option
end
```
- `def parse_option`：ユーザーが指定したオプションを解析するメソッド
- `option = ARGV.getopts('l', 'w', 'c')`：**オプションの取得**。`ARGV.getopts`で'-l', '-w', '-c'オプションを解析する。結果は`{ 'l' => true/false, 'w' => true/false, 'c' => true/false }`というハッシュになる
- `no_option = { 'l' => true, 'w' => true, 'c' => true }`：**デフォルトオプションの定義**。オプション指定がない場合のデフォルト動作を定義する。すべてのカウント（行・単語・バイト）を表示する。
- `option.value?(true) ? option : no_option`：**三項演算子による分岐**。`option.value?(true)`で、オプションのいずれかが`true`かを確認する。
  - `true`：ユーザーがオプションを指定した → `option`を返す
  - `false`：オプション指定なし → `no_option`（デフォルト）を返す

### 5. build_contents_by_file_nameメソッド（ファイル内容の構築）
```ruby
def build_contents_by_file_name
  if ARGV.size.zero?
    content = $stdin.read
    { '' => content }
  else
    contents = ARGV.map { |file_name| File.read(file_name) }
    ARGV.zip(contents).to_h
  end
end
```
- `def build_contents_by_file_name`：ファイル名をキー、ファイル内容を値とするハッシュを作成するメソッド。メソッド名が「何を構築するか」を明確に示している
- `if ARGV.size.zero?`：**条件分岐の開始**。`ARGV`（コマンドライン引数）が空（ファイル名の指定なし）かどうかを判定する
- `content = $stdin.read`：**標準入力の読み込み**。`$stdin.read`で標準入力から全ての内容を読み込み、変数`content`に代入する。パイプからの入力に対応する。
- `{ '' => content }`：**ハッシュを返す**。標準入力の場合、ファイル名は空文字`''`とし、`{ '' => content }`というハッシュを返す
- `contents = ARGV.map { |file_name| File.read(file_name) }`：**ファイル内容を配列にまとめる**：`ARGV`の各ファイル名に対して`File.read`でファイル内容を読み込み、内容の配列`contents`を作成する
- `ARGV.zip(contents).to_h`：**ハッシュに変換**
  - `ARGVzip(contents)`：ファイル名の配列とファイルの内容の配列を組み合わせて、`[['file1.txt', '内容1'], ['file2.txt', '内容2']]`のような配列を作成する
  - `.to_h`：それをハッシュ`{ 'file1.txt' => '内容1', 'file2.txt' => '内容2' }`に変換する
