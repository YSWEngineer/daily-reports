### オリジナルのHTMLの作成
自分で書いた HTMLファイルを配置し、ブラウザで表示させる。
#### 1. HTMLファイルを作成し、viエディタを開く
SSH接続されたターミナルで以下のコマンドを実行。
```shell
vi ~/index.html
```

#### 2. INSERT（挿入モード）に入る
`i`を押して、INSERTに切り替える。

#### 3. HTMLを入力
私は以下の内容を別のエディタで作成し、コピーアンドペースト。
```shell
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Yoshiwo's nginx practice</title>
</head>
<body>
  <h1>Hello, nginx!</h1>
  <p>nginx is working!</p>
  <p>FBC Practice: Understanding nginx basics</p>
</body>
</html>
```

#### 4. 保存して終了
1. `esc`を押す（`INSERT`が消える）
2. `:wq`と入力して`return`キーを押下（記録と終了）

#### 5. 確認
```shell
cat ~/index.html
```
上のコマンドを入力して HTML の内容が表示されればOK。

### HTMLファイルをnginxの公開ディレクトリに配置
作成した HTMLファイルを、nginx が読み込める場所にコピーする。
```shell
yoshiwo@tk2-262-40865:~$ sudo cp ~/index.html /var/www/html/
[sudo] password for yoshiwo:  # ここでもsudo権限を求められるので専用のパスワードを入力して、returnキーを押下
```
- `sudo`：管理者権限で実行
- `cp`：コピーコマンド
- `~/index.html`：先程作成したファイル
- `/var/www/html/`：nginx のディレクトリ

コピーできたか確認する。
```shell
yoshiwo@tk2-262-40865:~$ ls -l /var/www/html/
total 8
-rw-r--r-- 1 root root 230 Apr 29 14:51 index.html  # コピーされたindex.htmlファイルがある
-rw-r--r-- 1 root root 615 Apr 29 13:53 index.nginx-debian.html  # おそらくnginxのデフォルトページ
```
