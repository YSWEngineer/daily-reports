## 分かったこと・気付き
- レッスン3.1：貼り付けコマンド
    - `p` - 最後に削除された行をカーソルの後に貼り付けます。
        
        カーソルがある行の下の行（次の行）に貼り付けが行われます。
        
        p を押すたびに最後に削除された行を貼り付けることが可能です。
        
- レッスン3.2：置き換えコマンド
    - `r` の後に文字を打つことで、その打った文字に置き換えることができます。
        
        例）「諦めずに正しい努力を継族すること」という一文の「族」を「続」に置き換える場合は、
        
        ①先ず、「族」にカーソルを合わせます。
        
        ②Normal Modeであることを確認して、「r」を入力します。※「r」と入力するだけでreturn（Enter）は押しません。
        
        ③ローマ字日本語入力に切り替えて「続」を入力して、ここでreturn（Enter）を押します。
        
        ④「族」が「続」に置き換わります。
        
- レッスン3.3：変更コマンド
    - `ce` - 単語の末尾までを変更します。
        
        例）ssrei を study と正しい単語を打ちたい時は、
        
        ①カーソルを左から2番目の「s」に合わせます。
        
        ②「ce」と打つと「srei」の4文字が削除され、Insert Modeに切り替わります。
        
        ③「tudy」と正しく文字を入力します。
        
        ④もし、このまま単語を変更したい時は一度escキーを押してInsert ModeからNormal Modeに移行して、① ~ ③と同じ要領で行います。
        
    - `cc` - 「ce」と同じことを行全体に対して行います。つまり、行全体を削除して新しく文字・文章を入力することができます。
- レッスン3.4：「c」を使用したその他の変更
    - 変更オペレータは、削除と同じようにモーションを使用します。
    - 変更オペレータは、削除と同じような動作をします。その形式は
        
        `c [数値] モーション`
        
        モーションも同じで、「w」は単語、「$」は行末などといったものです。
        
        例）「The end of this line needs some help to make it like the second.」という文を「The end of this line needs to be corrected using the c$ command.」に書き直したい時は、
        
        ①カーソルを「some」の「s」に持っていきます。
        
        ②「c$」と入力すると「some help to make it like the second.」が消去され、Insert Modeに切り替わります。
        
        ③後はそのまま文字を入力します。
        
- レッスン3：要約
    1. すでに削除されたテキストを再配置するには、`p` を打ちます。これは削除されたテキストをカーソルの後に挿入します（行単位で削除されたのならば、カーソルのある次の行に挿入されます）。
    2. カーソルの位置にある文字を置き換えるには、`r` を打った後、それを置き換える文字を打ちます。
    3. 変更コマンドではカーソル位置からの特定のモーションで指定される終端までを変更することが可能です。例えば `ce` ならばカーソル位置から単語の終わりまで、`c$` ならば行の終わりまでを変更します。
    4. 変更コマンドの形式は、
        
        `c [数値] モーション`
        
        です。
        
- レッスン4.1：位置とファイルの情報
    - `control + g` でファイル名とカーソルがいる行番号が表示されます。
    - `G（大文字のG）` でファイルの最下行に移動します。
    - `gg` でファイルの先頭に移動します。
    - `行の番号 に G（大文字のG）` を打つとその行に移動します。
        
        例）619行目に移動したい時は、`619` と入力した後に `G` と入力します。
        
        ※指定した行は画面右下に表示されます。
        
- レッスン4.2：検索コマンド
    - 語句を検索するには `/` と、`検索する語句` を打ちます。
        
        ①Normal Modeであることを確認して「/」と打ちます。すると画面左下に「/」が表示されます。
        
        ②検索したい文字を入力して、return（Enter）を押します。
        
        ③検索にヒットした文字の頭文字にカーソルが移動します。
        
    - 検索した語句が他にもあるかどうかを確認したい時は `n` を打ちます。また、逆方向に語句を検索したい時は N（大文字のN）を打ちます。
    - 最初から逆方向に語句を検索する場合は、「/」 の代わりに `?` コマンドを使用します。
    - 検索する前の場所に戻るには control + o を打ちます。続けて入力することでさらに戻ることができます。逆に進みたい場合は control + i と打ちます。
- レッスン4.3：対応する括弧を検索
    - `%` - )、]、}などの対応する括弧を検索します。
        
        例）This ( is a test line with ('s, ['s ] and {'s } in it. )) このようなテキストがあった時に、「初めの『(』（丸括弧）」にカーソルを合わせた状態で「％」を打つとその丸括弧に対応した「一番最後の『)』（閉じ括弧）」を検索してくれます。
        
        この括弧の検索は、[、]、{、}にも対応されています。
        
        このような機能は括弧が一致していないプログラムをデバッグするのにとても役立ちます。
        
- レッスン4.4：間違いを変更する方法
    - 例）’old’ を ‘new’ に置換するには `:s/old/new/g` と打ちます。
        
        `:s/old/new` と入力した後にreturn（Enter）を押すと、その行で最初に見つかったものだけに置換が行われます。
        
    - `:s/old/new/g`と入力すると、oldからnewへの置換はその行で見つかった全ての箇所に対して行われます。
    - 複数行から見つかる文字のすべての箇所を変更するには
        
        `:#,#s/old/new/g` #,#には置き換える範囲の開始と終了の行番号を指定します。
        
        `:%s/old/new/g` ファイル全体で見つかるものに対して変更します。
        
        `:%s/old/new/gc` ファイル全体で見つかるものに対して、1つ1つ確認を取りながら変更します。
        
- レッスン4：要約
    1. `control + G` はファイルでの位置とファイルの詳細を表示します。
        
        `G` はファイルの最下行に移動します。
        
        `数値 G` はその行に移動します。
        
        `gg` はファイルの先頭行に移動します。
        
    2. `/` の後に語句をタイプすると前方に語句を検索します。
        
        `?` の後に語句をタイプすると後方に語句を検索します。
        
        検索の後の `n` は同じ方向の次の検索を、`N` は逆方向の検索をします。
        
        `control + O` は場所を前に移し、`control + I` は場所を次に移動します。
        
    3. (、)、[、]、{、若しくは } 常にカーソルがある状態で `%` をタイプすると対になる文字へ移動します。
    4. 現在行（カーソルが位置している行）の最初の old を new に置換する。                 `:s/old/new`
        
        現在行（カーソルが位置している行）の全ての old を new に置換する。                  `:s/old/new/g`
        
        2つの # 行の間で語句を置換する。                                                                                                    `:#,#s/old/new/g`
        
        ファイルの中のすべての検索語句を置換する。                                                                          `:%s/old/new/g`
        
        ‘c’ を加えると置換のたびに確認を求める。                                                                                   `:%s/old/new/gc`
        
- レッスン5.1：外部コマンドを実行する方法
    - `:!` の後に実行する外部コマンドを打ちます。
        
        例）`:!` と入力した後に、ls と打ってreturn（Enter）を押します。するとシェルプロンプトのようにディレクトリの一覧が表示されます。
        
    - この方法によってあらゆるコマンドを実行することができます。もちろん引数も与えられます。
    - 全ての `: コマンド`は `return（Enter）` を押すことで終了することができます。
- レッスン5.2：そのほかのファイルへ書き込み
    - `:w ファイル名` - ファイルへ変更を保存します。※w とファイル名の間には半角スペースが入ります。
    - `:!ls` - ディレクトリを確認することができます。
    - `:!rm ファイル名` - ファイルを消すことができます。
- レッスン5.3：選択した書き込み
    - `v モーション`と `:w ファイル名` でファイルの一部を保存します。
        
        ① `v` を押すと `Visual選択`が始まります。
        
        ②カーソルを動かすとテキストが強調表示されるので保存したい範囲を選択します。
        
        ③選択後、`:` を押すと、画面の最下部に `:’<,’>` が表示されます。
        
        ④ `w 保存するファイル名` を打ち込みます。※ `w`と `ファイル名` の間はスペースが入ります。
        
        ⑤return（Enter）を押す前に画面の最下部の表示が `:’<,’>w ファイル名` となっていることを確認します。
        
        ⑥確認後、return（Enter）を押すと、Vimはファイルに選択された内容を書き込みます。
        
        ⑦ `:!dir` または `:!ls` でファイルを確認できます。
        
    - `v` を押すと、`Visual選択`が始まります。カーソルを動かすことで、選択範囲を大きくも小さくもできます。さらにその選択範囲に対してオペレータを適用できます。例えば `d` はテキストを削除します。
- レッスン5.4：ファイルの取込と合併
    - `:r ファイル名` - ファイルの中身を挿入します。
    - カーソルの下の行にファイルの中身が挿入されます。
    - 外部コマンドの出力を読み込むこともできます。例えば、 `:r !ls` は lsコマンドの出力をカーソル以下に読み込みます。
- レッスン5：要約
    1. :!command によって外部コマンドを実行します。
        
        よく使う例：
        
        （Windows）                            （Unix）
        
        `:!dir`                                            `:!ls`                                      -      ディレクトリ内の一覧を見ることができます。
        
        `:!del ファイル名`                   `:!rm ファイル名`              -      ファイルを削除します。
        
    2. :w ファイル名 によってファイル名というファイルがディスクに書き込まれます。
    3. vモーションで :w ファイル名 とすると、ビジュアル選択行がファイルに保存されます。
    4. :r ファイル名 によりファイル名というファイルがディスクより取り込まれ、カーソル位置の下に挿入されます。
    5. :r !ls は lsコマンドの出力をカーソル位置から下に読み込みます。
- レッスン6.1：オープンコマンド
    - `o（小文字のo）` - カーソルの下に空行が開き、挿入モードに入ります。
    - `O（大文字のO）` - カーソルの上に空行が開き、挿入モードに入ります。
- レッスン6.2：追加コマンド
    - `a` - Insert Mordに切り替わり、カーソルの次の位置からテキストを追加できるようになります。
    - `eコマンド`は単語の終端にカーソルを移動させます。
    - a, i, Aは同じ挿入モードへ切り替わりますが、文字が挿入される位置だけが異なります。
        - a - カーソルの次の位置からテキストを追加できます。
        - i - カーソルの前の位置からテキストを追加できます。
        - A - テキストが書かれた行末からテキストを追加できます。
- レッスン6.3：その他の置換方法
    - `R（大文字のR）` - 1文字以上を置き換えることができます。
        
        ①置き換える対象の先頭に移動します。
        
        ② `R（大文字のR）`を押すと画面左下に `-- REPLACE --` と表示され、置き換えたい文字や記号を打つことで置換されます。
        
        ③置換モードを抜けるには、`escキー`を押します。
        
        ※置換モードは挿入モードに似ていますが、全てのタイプされた文字は既存の文字を削除します。
        
- レッスン6.4：テキストのコピーとペースト
    - テキストのコピーには `オペレータ y` を、ペーストには `p` を使います。
    - この時の `y` は yank（ぐいっと引っ張る。コピー。）を意味します。
    - `j$` - j は次の行に移動して、 $ は行末に移動します。
    - y をオペレータとして使うこともできます。`yw` は単語を1つ yank（コピー） します。`yy` は行を1つ yank し、`p` でその行を put します。
- レッスン6.5：オプションの設定
    - `:set ignorecase` または `:set ic（ignorecaseの短縮系）`は検索や置換の際に大文字/小文字の区別を無視する設定が行われます。。
    - `:set incsearch` または `:set is（incsearchの短縮系）`は検索フレーズのうち、部分的にマッチしている箇所を表示します。
    - `:set hlsearch` または `:set hls（hlsearchの短縮系）`はマッチする全てを強調表示します。
- レッスン6：要約
    1. o （小文字のo）を打つとカーソルの下の行を開けて、そこで挿入モードになります。
        
        O（大文字のO）を打つとカーソルの上の行で挿入モードになります。
        
    2. カーソル上の文字の次からテキストを追加するには a と打ちます。
        
        行末にテキストを挿入するには大文字 A を打ちます。
        
    3. e コマンドは単語の終端にカーソルを移動させます。
    4. y オペレータはテキストを yank（コピー）して、p はそれを put（ペースト）します。
    5. 大文字の R を打つと置換モードに入り、escキーで抜けることができます。
    6. `:set xxx` と打つと`オプション xxx` が設定されます。以下のオプションはそれぞれ短縮することが可能です。
        
        `ic` または `ignorecase`             検索時に大文字小文字の区別をしません。場合によっては大文字と小文字を区別したい場合があるかもしれません。そんな時はignorecaseオプションを無効にすることができて、大文字と小文字の区別ができます。
        
        `is` または `incsearch`                incremental search（インクリメンタルサーチ）を有効にするためのオプションです。incremental searchはゆうーざーが検索パターンを入力すると検索フレーズに部分マッチしている部分を表示します。
        
        `hls` または `hlsearch`                highlight search（ハイライトサーチ）を有効にするためのオプションです。highlight searchは検索結果のマッチ箇所を強調表示する機能です。
        
- レッスン7.1：オンラインヘルプコマンド
    - Vimには後半にわたるオンラインヘルプシステムがあります。
    - ヘルプを開始するには、これら3つのどれか1つを試してましょう。
        1. ヘルプキーを押します（もしあるならば）。
        2. F1キーを押します（もしあるならば）。
        3. `:help` と入力して、return（Enter）を押します。
    - ヘルプウィンドウのテキストを読むと、ヘルプの動作が理解できます。
    - ヘルプウィンドウに入っている時に、`control + W` を2回入力するとヘルプウィンドウから元の画面へジャンプすることができます。続けて control + W を2回入力するとヘルプウィンドウに戻ることができます。
    - `:q` を入力してreturn（Enter）を押すと、ヘルプウィンドウが閉じられます。
    - ヘルプウィンドウに入っている時に、:helpコマンドに引数を与えることで、あらゆるタイトルのヘルプを見付けることができます。
        1. `:help w`
        2. `:help c_CTRL-D`
        3. `:help insert-index`
        4. `:help user-manual`
- レッスン7.2：起動スクリプトの作成
    - VimにはViよりも多くの特徴を踏まえていますが、そのほとんどは初期状態にて使用不可となっています。より多くの特徴を使い始めるには `"vimrc"ファイル`を作成します。
        1. `"vimrc"` ファイルの編集を開始します。これはシステムに依存します。
            
            `:e ~/.vimrc`             UNIX 向け
            `:e ~/_vimrc`             Windows 向け
            
        2. ここでサンプルの "vimrc" を読み込みます。
            
            `:r $VIMRUNTIME/vimrc_example.vim`
            
        3. 以下のようにファイルへ書き込みます。
            
            `:w`
            
        
        次回 Vim を起動すると、色づけ構文が使えるようになるでしょう。
        この "vimrc" ファイルへ、お好みの設定を追加することができます。
        より多くの情報を得るには `:help vimrc-intro` とタイプします。
        
- レッスン7.3：補完
    - CTRL-D と TAB でコマンドラインを補完する
        1. 互換モードでないことを確認します： `:set nocp`
        2. 現在のディレクトリにあるファイルを `:!ls` か `:!dir` で確認します。
        3. コマンドの先頭をタイプします： `:e`
        4. CTRL-D を押すとVimは "e” から始まるコマンドの一覧を表示します。
        5. `d TAB` とタイプすると Vim は `:edit` というコマンド名を補完します。
        6. さらに空白と、既存のファイル名の始まりを加えます： `:edit FIL`
        7. `TAB` を押すと Vim は名前を補完します。(もし一つしか無かった場合)
