## 何を作るのか？（目的）
・ Unixコマンド`wc`を Ruby で再現するスクリプト（`wc.rb`）を作る
・テキストの「行数・単語数・バイト数」を集計して表示する

## 入力パターン（2種類に対応する）
### パターン1：ファイル指定
→ 引数にファイル名を渡す方法。ファイルが一つでも複数でも動く。
```shell
% ruby wc.rb 1.txt
% ruby wc.rb 1.txt 2.txt
```
### パターン2：パイプ（標準入力）
→ パイプで前のコマンドの出力を受け取る方法。
```shell
% ls -l | ruby wc.rb
% ruby ls.rb | ruby wc.rb
```

## オプション（6パターンすべてに対応する）
### 対応が必要なオプション一覧
- [ ] オプションなし → 行数・単語数・バイト数をすべて表示
- [ ] `-l`のみ → 行数だけ表示
- [ ] `-w`のみ → 単語数だけ表示
- [ ] `-c`のみ → バイト数だけ表示
- [ ] `-lw` / `-wc` / `-lc` → 指定した二つを表示
- [ ] `-lwc` → 全部表示（オプションなしと同じ）

表示順は「行数 → 単語数 → バイト数」で固定（本物の`wc`と同じ挙動で）。
オプションの指定順は関係ない。 

## 複数ファイルのときの動き
### total行を追加する
- [ ] ファイルが一つだけのときは、`total`行は不要
- [ ] ファイルが二つ以上のときは、各ファイルの結果の下に`total`（合計）行を表示する
```shell

# ファイルが二つ以上のとき
~/wc_practice % wc 1.txt 2.txt 
       3       9      45 1.txt

```

## 💡 本日の学び・気付き
### 1ファイルの行数・単語数・バイト数を表示する
#### ①行数・単語数・バイト数を表示する
Ruby でファイルの行数・単語数・バイト数を表示するには、以下のメソッドを使用する。
- `lines.count`（行数）
- `split.size`（単語数）
- `bytesize`（バイト数）

```ruby
content = File.read('1.txt')  # ファイルの中身を文字列として取り出す

lines = content.lines.count  # 行ごとに分割して数を数え、linesに代入する
words = content.split.size  # 単語ごとに分割して数を数え、wordsに代入する
bytes = content.bytesize  # （文字列の）バイト数を数え、bytesに代入する
puts lines  # 行数を表示する
puts words  # 単語数を表示する
puts bytes  # バイト数を表示する
```

#### ②行数・単語数・バイト数の表示を整形する
- 今回は最もシンプルな`rjust`（右揃えにする）メソッドを使用
- 1行にまとめて表示させたいので自動的に改行が付く`puts`ではなく、改行されない`print`を使用
- コマンドを実行するときに渡した引数からファイル名を取得する必要があるので、**組み込み定数**である`ARGV`を使用
```ruby
content = File.read(ARGV[0])  # ARGV[0]のファイルを読み込む
lines = content.lines.count  # 行数を数える
words = content.split.size   # 単語数を数える
bytes = content.bytesize     # バイト数を数える

print lines.to_s.rjust(8)  # 行数を右揃えで表示
print words.to_s.rjust(8)  # 単語数を右揃えで表示
print bytes.to_s.rjust(8)  # バイト数を右揃えで表示
print " #{ARGV[0]}"  # ファイル名を表示
print "\n"           # 改行

# 実行結果
~/wc_practice % ruby wc.rb 1.txt
       3       9      45 1.txt

# 本物のwcコマンド
~/wc_practice % wc 1.txt
       3       9      45 1.txt
```

### 復習：ARGVについて
#### なぜ書いていないのにARGVが使えるのか？
`ARGV`は Ruby が**自動的に用意してくれる特別な定数**だから。自分で書かなくても最初から使うことができる。

##### 仕組みをイメージ化してみる
```
ターミナルで入力
$ ruby wc.rb 1.txt
       ↑        ↑
  スクリプト名  引数

         ↓ Rubyが自動的に

ARGV = ["1.txt"]  ← Rubyが準備してくれる
```
##### 例：郵便配達のイメージ
```
自分（ターミナル）  →  手紙を投函（引数を渡す）
Ruby  →  自動的に郵便受け（ARGV）に入れてくれる
プログラム   →  郵便受け（ARGV）から取り出して使う
```
##### まとめ
```
$ ruby wc.rb 1.txt
              ↑
         Rubyが自動的に ARGV = ["1.txt"] に入れてくれる
              ↓
         プログラムの中で ARGV[0] で取り出せる
```
「**自分で書かなくてもRubyが準備してくれている**」と覚えておこう！
